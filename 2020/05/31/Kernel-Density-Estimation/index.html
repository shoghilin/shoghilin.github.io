Kernel Density Estimation (KDE)

什麼是KDE

KDE是採⽤無⺟數⽅法畫出⼀個觀察變數的機率密度函數，其中kernel在“非參數估計”的語境下，“kernel”是一個函數，用來提供權重。例如高斯函數 (Gaussian) 就是一個常用的kernel function。

Density plot 的特性

  • 歸⼀：線下 ⾯積和為 1
  • 對稱：K(-u) = K(u)

常⽤的 Kernel function

  • Gaussian (Normal dist)
  • Cosine

優缺點

優點：

  • 無母數方法，分布沒有假設

缺點：

  • 計算量大
